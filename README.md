# Power Plus LLC Website

Welcome to the official website repository for **Power Plus LLC**, a comprehensive energy and AI solutions company based in Sharjah, UAE.

🌐 **Live Website:** [powerplusllc.org](https://powerplusllc.org/)

## 🚀 Overview

Power Plus LLC specializes in:
- **Comprehensive Energy Excellence Program (CEEP):** A complete, integrated energy program combining lighting, solar, HVAC, and water efficiency.
- **Solar PPA:** Zero Capex solar solutions for commercial and industrial clients.
- **AI Access Control:** Seamless vehicle and pedestrian access control using advanced AI and existing cameras, removing the need for expensive dedicated hardware.
- **Generator Supply:** Reliable Cummins and Perkins diesel generators for heavy-duty applications across the UAE.

## 💻 Tech Stack

This project is built with a modern, high-performance web stack:
- **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** Cloudflare Pages

## 🛠️ Getting Started

To run this project locally, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/samuelalex3737/powerplus-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd powerplus-website
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Development Server

Start the local development server:
```bash
npm run dev
```
Open `http://localhost:5173` (or the port shown in your terminal) in your browser to view the site.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The compiled assets will be generated in the `dist` folder, ready for deployment.
